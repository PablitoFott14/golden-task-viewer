- **Background:** Dark navy (#0D1B2A) on all slides. All body text in white (#FFFFFF).
- **Accent color:** Use gold/amber (#F0A500) for slide titles, table headers, and any highlighted values.
- **Font:** Single sans-serif font throughout (e.g., Calibri or Arial). Title text 28–32pt, body text 16–18pt, table text 13–14pt.
