# Final Artifact Format Specification

## Output File
The final deliverable must be named: `final_insights.md`


## Required Sections (in order)

### 1. Current Liquid Cash
- A single labeled line stating Brandon's current available liquid cash
- Format: `**Current Liquid Cash:** $X,XXX.XX`

### 2. Remaining Amount Needed
- A single labeled line showing how much additional capital is required to reach the $25,000 launch budget
- Formula: $25,000.00 − Current Liquid Cash = Remaining Amount Needed
- Format: `**Remaining Amount Needed:** $X,XXX.XX`

### 3. Stocks to Liquidate

#### 3a. Recommended Positions to Sell
- A bullet point list including only the stocks that are part of the recommended liquidation combination withits name and its value after the 5% incrase.
- Format: `- STOCK — Value After +5%: $X,XXX.XX`


#### 3b. Estimated Funds Generated
- A single labeled line with the total estimated proceeds from selling all recommended positions
- Format: `**Estimated Funds Generated:** $X,XXX.XX`
- Follow immediately with a confirmation line indicating whether the combined liquid cash and estimated funds meet or fall short of the $25,000 goal
- Format: `**Total Available After Sales:** $X,XXX.XX — [meets / does not meet] the $25,000 launch budget`

### 4. Competitor Pricing Analysis

#### 4a. Stanley Quencher Pricing
- A single line stating Stanley Quencher Price
- Format: `**Stanley Quencher Price** $XX.XX`

#### 4b. Estimated Manufacturing Capacity
- State the budget allocated to production as 80% of $25,000
- Format: `**Production Budget (80%):** $X,XXX.XX`
- State the assumed production cost per unit (5% below the identified retail price) for each Stanley Quencher variant used as the reference
- Format: `**Production Cost per Unit:** $XX.XX (based on [Model/Size] retail price of $XX.XX)`
- State the estimated number of units that can be manufactured
- Format: `**Estimated Units Manufacturable:** X,XXX units`
